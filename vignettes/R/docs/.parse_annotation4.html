.parse_annotation4 {Daisy} R Documentation

Parse MS2 fragment annotations

Description

Usage

.parse_annotation4(lines);

Arguments

lines

Character vector where each element contains whitespace-separated values (expected format: peak_index, m/z value, annotation text).

Details

Input lines should contain at least three whitespace-separated columns. Second column is parsed as m/z, third column as annotation text.

Authors

xieguigang@metabolomics.ac.cn

Value

Dataframe containing: \itemize{ \item \code{mz} - Numeric vector of fragment m/z values \item \code{peak_anno} - Character vector of corresponding peak annotations }
