__peak_alignment {Daisy} R Documentation

(Internal) Perform peak alignment across samples

Description

Usage

__peak_alignment(xcms.out,
    max.rtwin = 15,
    workdir = './');

Arguments

xcms.out

Directory containing XCMS peak CSV files.

max.rtwin

Maximum retention time window (seconds) for peak matching.

workdir

Output directory for generated files (default: "./").

Details

Authors

xieguigang@metabolomics.ac.cn

Value

Invisibly returns NULL. Generates: \itemize{ \item Consolidated peak table (peaktable.csv) \item Retention time shift data (rtshifts.csv) \item Peak metadata (peakmeta.csv) \item Diagnostic plots (rtshifts.pdf, peakset.pdf) }
