export_report {Daisy} R Documentation

Export Metabolomics Annotation Report with Spectrum Alignment Visuals This function processes mass spectrometry annotation results from multiple sample files, merges them into a consolidated report, generates spectrum alignment mirror plots for each hit, and exports the results to specified directory. The function handles file I/O operations and data processing in a pipeline workflow.

Description

Usage

export_report(files,
    export.dir = './');

Arguments

files

Character vector of file paths containing sample annotation results. Each path should point to a sample directory containing `result.csv`2,6.

export.dir

Output directory path for saving consolidated report and visualizations. Defaults to current working directory (`"./"`).

Details

Authors

xieguigang

Value

Invisibly returns `NULL`. Primary outputs are: - `exportdir/annotationresult.csv`: Consolidated report table - `exportdir/spectralaligns/*`: Spectrum alignment plots (SVG/PNG)
