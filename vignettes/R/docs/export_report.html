export_report {Daisy} R Documentation

Export Metabolomics Annotation Report with Spectrum Alignment Visuals

Description

Usage

export_report(files,
    export.dir = './');

Arguments

files

Character vector of file paths containing sample annotation results. Each path should point to a sample directory containing `result.csv`2,6.

export.dir

Output directory path for saving consolidated report and visualizations. Defaults to current working directory (`"./"`).

Details

Authors

xieguigang

Value

Invisibly returns `NULL`. Primary outputs are: - `exportdir/annotationresult.csv`: Consolidated report table - `exportdir/spectralaligns/*`: Spectrum alignment plots (SVG/PNG)

Examples

\dontrun{ # Basic usage export_report(files = c("sample1", "sample2")) # Custom output directory export_report( files = list.dirs("raw_data", recursive = FALSE), export_dir = "processed_results" )}
