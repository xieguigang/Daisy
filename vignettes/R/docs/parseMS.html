parseMS {Daisy} R Documentation

Parse MS2 spectrum data from text block

Description

Usage

parseMS(block);

Arguments

block

Character vector where each element contains a pair of whitespace-separated numeric values (m/z and intensity).

Details

Each line in the input block should contain exactly two numeric values separated by whitespace. The first value is parsed as m/z, the second as intensity. Non-numeric entries will result in NA values.

Authors

xieguigang@metabolomics.ac.cn

Value

A dataframe containing two columns: \itemize{ \item \code{mz} - Numeric vector of fragment m/z values \item \code{into} - Numeric vector of corresponding intensity values }

Examples

\dontrun{ block <- c("100.0 500", "150.2 2500") parseMS(block) }
