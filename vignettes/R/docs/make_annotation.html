make_annotation {Daisy} R Documentation

Run metabolite annotation search

Description

Usage

make_annotation(files, peakfile,
    libtype = [-1],
    ms1ppm = 15,
    export.dir = './',
    library.dir = NULL,
    n.threads = 8,
    waters = FALSE,
    metadna = TRUE,
    do.plot = TRUE,
    id.range = NULL,
    debug = FALSE);

Arguments

peakfile

the file path to the xcms peaktable file.

libtype

the ion polarity mode of the rawdata files, should be an integer vector of 1 for positive and -1 for negative raw data.

export.dir

a character vector of the directory path for output the annotation result files.

n.threads

the number of the cpu threads for run for the parallel compute.

Details

the input raw data files could be in data formats of: 1. open source: mzXML, mzML 2. panomix mzkit format: mzPack, PeakMs2

Authors

xieguigang@metabolomics.ac.cn

Value

This function has no value returns.
