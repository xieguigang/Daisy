build_cfmid3_library {Daisy} R Documentation

Build Reference Spectrum Library from CFM-ID 3.0 Outputs

Description

Usage

build_cfmid3_library(cfmid, libfile);

Arguments

cfmid

Character string specifying the directory path containing CFM-ID 3.0 output files. Files must be in TXT format with predicted spectra.

libfile

Character string defining the output file path for the compiled spectral library. Recommended file extension is `.spa` for compatibility with `mzkit` packages.

Details

Authors

xieguigang@metabolomics.ac.cn

Value

None (invisible `NULL`). The function's primary output is a binary library file saved at the location specified by `libfile`.

Examples

\dontrun{ build_cfmid3_library( cfmid = "path/to/cfm_id_outputs", libfile = "reference_library.spa" )}
