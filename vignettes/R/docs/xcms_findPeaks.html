xcms_findPeaks {Daisy} R Documentation

Perform peak detection using XCMS

Description

Usage

xcms_findPeaks(raw.files,
    workdir = './',
    docker = NULL);

Arguments

raw.files

Character vector of raw data files (mzXML/mzML format).

workdir

Output directory for peak CSV files (default: "./").

docker

Docker image ID for execution environment (optional).

Details

Uses either: \enumerate{ \item Docker container execution (if \code{docker} provided) \item Native R environment via \code{REnv::rlang_interop()} }

Authors

xieguigang@metabolomics.ac.cn

Value

Invisibly returns NULL. Outputs CSV peak files in \code{workdir}, one per input file (named using original basename).
